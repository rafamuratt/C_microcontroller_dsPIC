# C-Language 
Control a generic 16x2 LCD with the microcontroller dsPIC30F4013 using IDE's LCD library (MiKroC Pro for dsPIC - free version)
Basic application: 
Reads analog input and display its value (in % + bargraph with custom character). It also send the readed data over UART serial connection.
