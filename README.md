# C-Language 
Control a generic 16x2 LCD with dsPIC dsPIC30F4013 using IDE's LCD library (MiKroC Pro for dsPIC - free version)
Basic application: reads analog input and display its value (in % + bargraph with custom character).

Example:

Level: 98%
█ █ █ █ █ █ █ █ █ █
